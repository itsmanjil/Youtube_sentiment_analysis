# Frontend (React)

This is the React dashboard for the YouTube Sentiment Analysis project.

## Prerequisites
- Node.js 16+ (18+ recommended)
- Backend API running at http://127.0.0.1:8000

## Setup
```bash
cd frontend
npm install
npm start
```

## Configuration
The API base URL is defined in `frontend/src/axios.js`.

## Build
```bash
npm run build
```
